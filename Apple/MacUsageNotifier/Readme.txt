Hello,

This is the script just for showing macOS notifications with basic macOS stats in chosen time intervals (default set to 60s).
Shown information:
- uptime,
- CPU usage,
- RAM used,
- RAM Free,
- Swap usage.

You can see how these notifications look in the image.png file.
