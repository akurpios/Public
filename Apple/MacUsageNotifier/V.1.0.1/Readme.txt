Update:
1. Follow the Uninstallation guide for your current version,
2. Follow the Intallation guide for the current version.

Installation:
1. Download MacUserNotifier.zip to the Downloads folder,
2. Extract,
3. Run Install.command,

Uninstallation:
1. Open Finder and go to /Users/Shared/Scripts/Kurpios/MacUserNotifier,
2. Run Uninstall.command

Change interval of notification:
1. Open terminal,
2. Type:
  open ~/Library/LaunchAgents/com.kurpios.MacUsageNotifier.Schedule.plist
  some TextEdit app should open.
3. Change the value of StartInterval (in seconds) and save file,
4. Open Finder and go to /Users/Shared/Scripts/Kurpios/MacUserNotifier,
5. Run ReloadSettings.command
